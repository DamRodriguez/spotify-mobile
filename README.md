# 🎵 Spotify Mobile 

Clon de la aplicación de **Spotify** desarrollado con **React Native** y **Expo**.<br>
Este proyecto se centra en la fluidez de la interfaz de usuario, la gestión de estados complejos y la optimización del rendimiento en listas de reproducción extensas.
